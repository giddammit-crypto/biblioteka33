## 2024-05-23 - [Localization Pattern]
**Learning:** This theme ('city-library') uses Russian as the hardcoded source language in PHP templates (e.g., `_e('Russian')`) and lacks a `languages` directory.
**Action:** When adding accessible labels (aria-label), use Russian instead of English to match the site's hardcoded language and ensure screen readers speak the correct language for the user base, avoiding mixed-language interfaces.
